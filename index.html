Para usar la app con base de datos compartida, ingresa tu configuración de Firebase. Solo se hace una vez — queda guardada en este navegador.
Paso 1 · Crear proyecto
Ve a console.firebase.google.com → Crear proyecto → nombre: shark-tank-ver
Paso 2 · Habilitar Firestore
Build → Firestore Database → Crear → modo producción → elige región us-central
Paso 3 · Reglas de acceso
Firestore → Reglas → pega: allow read, write: if true; → Publicar
Paso 4 · Obtener credenciales
Project Settings → General → Your apps → Web app (</>) → registra la app → copia los valores del objeto firebaseConfig
Verifica los datos — no se pudo conectar a Firebase
Los datos de configuración se guardan en localStorage de este navegador.
No se comparten ni se envían a ningún servidor externo.
